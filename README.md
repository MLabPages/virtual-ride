# バーチャルライド (virtual-ride)

ルームバイクを漕ぐと、街並みの映像が**速度に連動して**流れる web アプリです。
退屈しがちな室内トレーニングを、観光気分で楽しめるようにします。

公開URL: https://mlabpages.github.io/virtual-ride/

## できること

- **速度連動の映像再生** — 速く漕ぐほど映像も速く流れ、止まると映像も止まります
- **2つの計測方法**
  - 📷 **カメラ計測** — スマホをバイクの横に置くと、ペダル(足元)の周期的な動きを
    検出して回転数(rpm)→速度に換算します(追加機器なし)
  - 🎚 **手動** — 画面のスライダーで速度を設定します
- **6つの街シーン** — 街なかライド / 夕暮れの旧市街 / 森の道 / 郊外ドライブ /
  街をドライブ / 街を散歩
- 速度・回転数・距離・時間の表示、画面消灯の防止、全画面表示

## プライバシー・セキュリティ

- **カメラ映像は端末のブラウザ内だけで処理し、保存も送信もしません。**
  このアプリはサーバーを持たず、外部への通信を一切行いません(数値の計算のみ)。
- 設定(ギアの重さなど)は端末の localStorage にのみ保存されます。
- 通信が HTTPS になるため、カメラ機能は公開URL(GitHub Pages)上で正しく動作します。

## 映像のライセンス

映像はすべて [Pexels](https://www.pexels.com) の無料ライセンス
(商用利用可・クレジット表記不要)の素材を使用しています。
各映像の出典は、アプリ内の「設定 → 映像クレジット一覧」で確認できます。

## 使い方(スマホ)

1. 上記の公開URLをスマホのブラウザで開く
2. 「📷 カメラ計測」を選び、カメラの使用を許可する
3. スマホをバイクの横に置き、ペダル(足元)が画面に映るように角度を合わせる
4. 漕ぎ始めると映像が流れ出します

うまく検出できないときは、⚙ 設定で「検出感度」や「ギアの重さ」「足の映り方」を調整してください。

## 開発メモ

ビルド不要の静的サイトです(`index.html` / `app.js` / `styles.css` / `videos/`)。
`main` ブランチへの push で GitHub Pages に自動デプロイされます
(`.github/workflows/pages.yml`)。
